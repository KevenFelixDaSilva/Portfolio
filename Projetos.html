App Mobile

Criação do meu primeiro projeto, desenvolvido em
Javascript e React Native para a conclusão do curso técnico.
O projeto foi realizado em grupo, onde cada membro foi
designado para uma função específica. Tive a oportunidade
de desenvolver não apenas habilidades hards skills, mas
também habilidades softs skills. Fui responsável pelo
desenvolvimento do back-end, aplicando o padrão DDD, como
ORM utilizei Sequelize e para solicitações HTTP Axios.
